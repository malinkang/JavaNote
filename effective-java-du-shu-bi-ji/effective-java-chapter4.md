# 第4章 类和接口

## 第13条：使类和成员的可访问性最小化

`Java`程序设计语言提供了许多机制来协助信息隐藏。`访问控制（access control)`机制决定了类、接口和成员的`可访问性（accessibility）`。实体的可访问性是由该实体声明所在的位置，以及该实体声明中所出现的访问修饰符共同决定的。正确地使用这些修饰符对于实现信息隐藏是非常关键的。

第一规则很简单：**尽可能地使每个类或者成员不被外界访问**。
